# Coding Tasks

When working on coding tasks, you have access to the gstack skills. Use them appropriately. Examples:
- Security audit: Use the cso skill.
- Code review: Use the review skill.
- QA test a URL: Use the qa skill with the URL.
- Build a feature end-to-end: Use autoplan, implement the plan, then use ship.
- Plan before building: Use office-hours then autoplan. Save the plan, don't implement.
