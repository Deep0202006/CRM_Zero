# 03 Design Reference Matrix: Operational Intelligence Workspace

This matrix maps design influences across CRM Zero routes and components.

| Influence Target | Blend Weight | Design Attributes | Application Scope |
| :--- | :--- | :--- | :--- |
| **Attio** | 35% | Contextual record inspection, fast relationship links, right-side preview drawer | Record detail pages (`/onboarding`, `/support`), Persistent Inspector |
| **Linear** | 25% | Keyboard shortcuts (`Cmd + K`), compact control heights (`32-36px`), zero noise lists | Navigation shell, Command palette, List toolbars, Filter chips |
| **Stripe Dashboard** | 10% | Predictable list/detail split, calm status badges, tabular numbers | Finance/payment logs, Attendance reports, Table cells |
| **Vercel Geist** | 10% | Typography hierarchy (`Inter`/`Geist`), strict spacing grid, high-contrast borders | Typography scale, Form headers, Border tokens |
| **Plane** | 10% | Multi-view toggles (Kanban/Table), reusable filters | Pipeline workspace (`/onboarding`), Task queues |
| **Twenty CRM** | 5% | Open-source CRM record identity standards | Lead identity header `{Name} (@{Username}) - {Phone}` |
