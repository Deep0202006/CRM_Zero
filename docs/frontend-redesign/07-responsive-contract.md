# 07 Responsive Contract & Breakpoints

## 1. Breakpoint Grid
- `mobile-small`: 360px
- `mobile`: 640px
- `tablet`: 768px
- `desktop-small`: 1024px
- `desktop`: 1280px
- `desktop-wide`: 1536px

## 2. Padding Scale
- Below 640px: 16px padding
- 640–767px: 20px padding
- 768–1279px: 24px padding
- 1280px and above: 32px padding

## 3. Mobile Navigation & Touch Specifications
- Minimum touch target for primary controls: `44px x 44px`.
- Desktop sidebar collapses into a mobile slide-over drawer triggered from the top utility bar.
- Long text fields and forms stack to single column on viewports under 768px.
