# 08 Visual QA Checklist & Verification Matrix

## Target Viewports
- [ ] `1440 x 900` (Desktop Large)
- [ ] `1280 x 800` (Desktop Standard)
- [ ] `1024 x 768` (Tablet Landscape)
- [ ] `768 x 1024` (Tablet Portrait)
- [ ] `390 x 844` (Mobile iPhone)
- [ ] `360 x 800` (Mobile Android)

## Checklist Verification Criteria
1. **Identity Standard:** Identity vectors strictly render `{Name} (@{Username}) - {Phone}`.
2. **Single Completion Action:** Only ONE action button labeled **"Done"** exists on follow-up and task cards.
3. **Typography & Tabular Numerals:** Table metrics, timestamps, and deal figures align using tabular numbers.
4. **System States:** Complete Skeleton loading, Error Boundaries, Empty States, and Permission Denied fallbacks.
5. **No Layout Overflow:** Zero horizontal scrollbars on main application layouts.
